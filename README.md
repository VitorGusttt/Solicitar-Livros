# Sistema de Solicitações de Livros para a Biblioteca

Este projeto está sendo desenvolvido para aprimorar os meus conhecimentos em MySQL, JavaScript e PHP. Além disso, estou utilizando a API do Google Livros. O objetivo principal é criar um sistema de solicitações de livros para a biblioteca da escola onde estudo. Isso se tornou necessário porque o sistema existente na escola está desativado no momento.

## Recursos Principais

- **Autenticação de Usuários:** Os usuários podem se registrar e fazer login no sistema.

- **Cadastro de Livros Desejados:** Os usuários podem cadastrar os livros que desejam ter disponíveis na biblioteca.

- **Avaliação de Livros:** Os livros cadastrados terão um sistema de avaliação com botões de "Like" e "Dislike" para que os usuários possam expressar sua opinião sobre os livros.

## Tecnologias Utilizadas

- **MySQL:** O banco de dados é gerenciado com MySQL para armazenar informações sobre usuários, livros e avaliações.

- **JavaScript:** A lógica de front-end é construída com JavaScript para melhorar a experiência do usuário.

- **PHP:** O back-end é desenvolvido em PHP para lidar com a lógica de autenticação, registro, cadastro de livros e avaliações.

- **API do Google Livros:** Utilizo a API do Google Livros para obter informações sobre os livros, incluindo capas e detalhes.
